## 快速开始

最小推理示例（完整 API 见 [任务推理API.md](任务推理API.md)）：

```rust
use yolo26_rs::{FilterOption, Image, detect};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let image = Image::from_file("examples/bus.jpg")?;
    let model = detect::Model::from_file("yolo26s.pt")?;
    let detections = model.predict(&image, &FilterOption::default())?;
    Ok(())
}
```

构建/验证：

```bash
cargo build --release
cargo build --release --features train
```

本机没有 CUDA toolkit 时不要把 `--all-features` 当作默认验证命令；`cuda` feature 会构建 `cudarc` 并要求 `nvcc`。

## 与官方 Ultralytics 的关系

- 官方 Ultralytics 是 Python/PyTorch 生态，直接加载 `.pt`，提供完整训练器、增强、验证、导出和部署工具。
- 本 crate 是 Rust/Candle 实现，以官方 `.pt` 为主要权重格式（默认 `pt` feature），推理和训练路径都在 Rust 侧运行；训练后可用 `save_pt` 写回官方 PyTorch 可读 `.pt`。
- YOLOE export 语义与官方一致：动态 prompts 必须在 export 前 bake，export 后模型不再接受新的 text/visual prompts。
