# 旋转目标检测推理（obb）

本篇演示如何用 `obb` 任务根做 YOLO26 旋转框（Oriented Bounding Box）检测：每个目标用一个带角度的旋转矩形表示，常用于航空/遥感（DOTA）、文档、任意方向排列的物体。`BBox` 字段是中心点 + 宽高 + 角度，与 detect 的轴对齐框不同。

完整 API 参考见 [任务推理API.md](../任务推理API.md)。

## 何时用

- 目标有明显方向（船只、车辆、文字行、细胞），轴对齐框会重叠或浪费面积。
- DOTA 风格的旋转框检测任务。
- 需要把旋转框转回轴对齐框（`axis_aligned_bbox`）或可视化四角点。

## 准备

- **Feature**：需 `--features obb`（不在默认 feature 里）。
- **权重**：OBB 官方 `.pt` 权重名形如 `yolo26s-obb.pt`（`-obb` 后缀）。
- **默认类别**：DOTA（15 类）。`labels_count` 从 checkpoint head shape 推断；类名表 `default_labels::DOTA` 需启用 `default_labels` feature（默认关），或用 `.pt` 权重的 `pt_loader::load_pt_metadata().names` 读取。

```bash
cargo build --release
```

## Rust 示例

```rust
use yolo26_rs::{FilterOption, Image, obb};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let image = Image::from_file("examples/boats.jpg")?;
    let model = obb::Model::from_file("yolo26s-obb.pt")?;
    let dets = model.predict(&image, &FilterOption::default())?;
    for d in &dets {
        println!(
            "class={} conf={:.2} bbox={:?}",
            d.class_id, d.confidence, d.bbox,
        );
        let _ = d.bbox.axis_aligned_bbox();
    }
    Ok(())
}
```

## 对应官方 Python 代码

```python
from ultralytics import YOLO

model = YOLO("yolo26s-obb.pt")
results = model.predict("examples/boats.jpg", imgsz=640, conf=0.25)
for r in results:
    for i, b in enumerate(r.obb):
        print(b.cls.item(), b.conf.item(), b.xywhr[i].tolist())
```

## API 与配置项详解

`obb::config_builder()` 默认输入 640、类别数 **DOTA**。builder 方法与 detect 完全一致（见 [物体检测.md](物体检测.md)）。

### 模型与预测

| API | 说明 |
| --- | --- |
| `obb::Model::from_file(path)` | 自动识别 `.pt` / `.safetensors` 并加载旋转框模型。 |
| `obb::Model::from_pt_file(path, config)` | 从官方 `.pt` 加载。 |
| `obb::Model::from_safetensors(weights, config)` | 从 `.safetensors` 字节加载。 |
| `model.forward_tensor(&input)` | raw forward，返回 `[B, A, nc+5+1]`（含角度）。 |
| `model.predict(&image, &filter)` | 端到端推理，返回 `Vec<obb::Prediction>`。 |

### `obb::BBox`（旋转框，注意与 `crate::BBox` 区分）

| 字段/方法 | 说明 |
| --- | --- |
| `center_x: f32` / `center_y: f32` | 旋转矩形中心（源图坐标）。 |
| `width: f32` / `height: f32` | 旋转前矩形的宽高。 |
| `angle: f32` | 旋转角（弧度）。 |
| `axis_aligned_bbox()` | 返回外接轴对齐 `crate::BBox`。 |
| `translate(dx, dy)` | 平移中心，角度不变。 |

### `obb::Prediction` 字段

- `bbox: obb::BBox`、`confidence: f32`、`class_id: u32`。

## 与官方的差异 / 注意事项

- 官方 `obb.xywhr` 是 tensor；本 crate 用强类型 `obb::BBox { center_x, center_y, width, height, angle }`。
- 可视化旋转框需自己用 `angle.sin_cos()` 算四个角点。
- OBB 评估用 rotated/probIoU mAP（见 [训练.md](../训练.md) 的 eval 段）。
